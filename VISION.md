The goal is to be an all in one application for smooth management of rental properties.
Every aspect of the property ( apartment|flat|condo|villa ) renting process will be addressed.

The pain point of leases will be handled. Owners will be able to pick and choose which clauses are added to the lease.
The lease will be allowed to be automatically printable with stamp paper.
Signatures and other tasks can be collected online, streamling the entire rental property leasing process.
The target location for this app is India.

We want to create an Android application.
We want to create an IOS application.
We want to create a web application.

This is going to be a very simple easy to use application for everyone.
No extra bloat. No extra unnecasary features.
Just what is going to be necassary for this purpose.
